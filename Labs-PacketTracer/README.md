## 🛠️ Requirements

- Cisco Packet Tracer 8.x (free from [Cisco NetAcad](https://www.netacad.com/))
- Basic knowledge of IOS CLI
- Completion of Module 01–05 notes recommended

---

## 📝 How to Use These Labs

1. Open Packet Tracer
2. Build the topology shown in each lab's **Topology** section
3. Follow the **Step-by-Step Configuration** commands
4. Use the **Verification** commands to confirm it's working
5. Answer the **Review Questions** at the end
6. Update status above from `⬜` to `✅` when done
