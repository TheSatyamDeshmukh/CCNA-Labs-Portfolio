## 🎯 What's Inside Each Module

### Module 01 — OSI Model
All 7 layers with functions, protocols, PDU names, ASCII diagrams, encapsulation/de-encapsulation flow, real-world device mapping, and 7 exam tips.

### Module 02 — TCP/IP Model
4-layer model vs OSI comparison, full protocol-port reference table (30+ protocols), ARP/ICMP/DNS/DHCP deep dives with packet flows, and 7 exam tips.

### Module 03 — IP Addressing
IPv4 binary structure, address classes, RFC 1918 private ranges, special addresses (APIPA, loopback), subnet masks, CIDR, magic number subnetting method with worked examples, VLSM, and complete cheat sheet.

### Module 04 — IPv6
Address format, abbreviation rules, all address types (GUA/ULA/LLA/Multicast/Anycast), EUI-64 step-by-step, SLAAC, DHCPv6, ICMPv6, NDP replacing ARP, DAD, transition mechanisms (dual-stack, tunneling, NAT64), and Cisco IOS config.

### Module 05 — TCP vs UDP
TCP/UDP header breakdowns, 3-way handshake, 4-way termination, flow control and windowing, congestion control (Slow Start, CA), all well-known ports table (30+ ports), and decision framework for choosing TCP vs UDP.

### Module 06 — Switching & VLANs ⭐ Largest Module
Ethernet frames, MAC addresses, switch forwarding logic, CAM table, duplex mismatch, collision vs broadcast domains, VLANs with full config, 802.1Q trunking and frame tagging, DTP modes and security, Inter-VLAN routing (ROAS + L3 SVI), VTP danger scenarios, STP 802.1D election process (all 4 steps), RSTP 802.1w, PortFast/BPDU Guard/Root Guard/Loop Guard, EtherChannel (LACP/PAgP/Static), Port Security (all violation modes), and complete switch hardening.

### Module 07 — Routing Fundamentals ⭐ Largest Module
Router packet forwarding step-by-step, routing table anatomy, longest prefix match, administrative distance complete table, static routing (4 methods), floating static routes, default routes, distance vector vs link state comparison, OSPF (cost, areas, DR/BDR, neighbor states, full config, wildcard masks, troubleshooting), EIGRP (DUAL algorithm, Successor/FS/FD/RD, unequal-cost LB, named mode), RIP v1 vs v2, route redistribution, and routing troubleshooting.

### Module 08 — ACLs & Network Security
CIA Triad, threat types, standard ACLs (wildcard masks, placement), extended ACLs (protocol/port matching, established keyword), named ACLs (sequence editing), ACL placement golden rules, AAA framework, RADIUS vs TACACS+ deep comparison, port security (all violation modes + recovery), DHCP Snooping (trusted/untrusted ports), Dynamic ARP Inspection, L2 attack mitigations, IPSec VPN phases, and 15 exam tips.

### Module 09 — WAN & SD-WAN
WAN terminology (CPE/DCE/DTE/CO/POP), topology types (hub-spoke/partial/full mesh), leased lines, HDLC vs PPP (PAP vs CHAP authentication), PPPoE DORA process + config, MPLS label structure/LER/LSR/L3VPN, Metro Ethernet, GRE tunnels (full config, MTU), IPSec phases, SD-WAN principles, Cisco SD-WAN (vManage/vSmart/vBond/vEdge), OMP, Zero-Touch Provisioning, application-aware routing, and WAN troubleshooting.

### Module 10 — IP Services
NAT types (static/dynamic/PAT with full config), NAT terminology (Inside Local/Global/Outside), DHCP DORA process, Cisco DHCP server config, ip helper-address explanation, NTP stratum hierarchy + authenticated config, SNMP versions (v1/v2c/v3 with authPriv), MIB/OID, syslog levels 0-7 + config, FTP vs TFTP comparison, QoS requirements for VoIP/video, DSCP markings, LLQ, policing vs shaping, and MQC full config.

### Module 11 — Network Automation
Why automation matters, traditional vs controller-based networking, the 3 planes (data/control/management), SDN northbound/southbound APIs, OpenFlow, Cisco DNA Center and IBN, SD-Access (VXLAN/LISP/ISE), REST APIs (all HTTP methods, status codes), Python REST API example, JSON/XML/YAML formats with examples, NETCONF (RFC 6241, SSH port 830, XML), RESTCONF (HTTPS, JSON/XML), YANG data models, Ansible (agentless, playbook example), Puppet and Chef comparison.

### Module 12 — Wireless Networking
Half-duplex nature of wireless, CSMA/CA vs CSMA/CD, hidden node problem + RTS/CTS, RF fundamentals (dBm, RSSI, SNR, path loss, MIMO), 802.11 standards table (a/b/g/n/ac/ax), OFDM/OFDMA/MU-MIMO/BSS Coloring, 2.4 GHz (channels 1/6/11) vs 5 GHz (24+ channels), BSS/ESS/IBSS/DS, autonomous vs lightweight vs cloud architectures, split-MAC, FlexConnect, CAPWAP (UDP 5246 control + UDP 5247 data), AP discovery (DHCP Option 43), WEP/WPA/WPA2/WPA3 evolution, SAE, OWE, PMF, 802.1X EAP flow, EAP-TLS vs PEAP vs EAP-FAST, WLC WLAN config, RRM/TPC, and wireless troubleshooting.
